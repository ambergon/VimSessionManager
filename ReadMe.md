VimSessionManager
=================
快適なセッション管理を手伝うプラグインです。
セッションファイルが一箇所に管理できないことを不満に思い、作成しました。

Usage:
------
```
:SessionSave {file_name}
```
規定のディレクトリにmksessionファイルを保存します。
同名ファイルが存在する場合上書きするか確認します。

```
:SessionLoad {file_name}
```
規定ディレクトリからセッションファイルを読み込みます。
